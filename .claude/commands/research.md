# /research — Web research on a topic

Run web research and produce a structured summary with sources.

## Usage

`/research [topic]` — e.g., `/research matcha wholesale pricing US 2026`, `/research competitor cafe suppliers`

If no topic is provided, ask what to research.

## Steps

1. **Search:**
   - Run 2-3 web searches with different angles on the topic
   - Focus on recent, relevant results (prioritize 2025-2026 data)

2. **Analyze:**
   - Read the most promising results
   - Extract key data points, trends, and insights
   - Note any conflicting information between sources

3. **Output a structured summary:**
   ```
   ## Research: [Topic]
   **Date:** [today's date]

   ### Key Findings
   - finding 1
   - finding 2
   - finding 3

   ### Details
   [expanded analysis organized by sub-topic]

   ### Sources
   - [Source title](URL) — what it contributed
   ```

4. **Ask if the user wants to:**
   - Save this as a note in the Obsidian vault
   - Dig deeper into any specific finding
